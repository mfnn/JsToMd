# exportExecl

> 工具函数说明： 二进制流转Execl


| 参数名 | 参数类型 | 参数说明 |
| ------ | -------- | ---- |
|content|字符串|二进制流内容|
|fileName|字符串|Execl表名|

    
返回： {file} Execl表格
# buildTree

| 参数名 | 参数类型 | 参数说明 |
| ------ | -------- | ---- |
|array|数组|数组|
|id_key|字符串|树形ID键名|
|parentId_key|字符串|树形父级ID键名|
